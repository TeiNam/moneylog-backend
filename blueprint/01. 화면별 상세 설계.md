# 📱 화면별 상세 설계

## 1. 일반 영역 (통합 뷰)

### 1-1. 홈 (대시보드)

```
┌──────────────────────────────┐
│ 2026년 3월                 ▼ │
├──────────────────────────────┤
│  이번달 요약                  │
│  ┌────────┬────────┬───────┐│
│  │ 수입    │ 지출    │ 잔액  ││
│  │3,200,000│1,850,000│1,350K ││
│  └────────┴────────┴───────┘│
│                              │
│  목표 달성률        [60%] ███░│
│  저축률             [42%] ██░░│
│  무지출 성공일       8일      │
├──────────────────────────────┤
│  소비성 지출 TOP 5            │
│  🍽 식비         520,000  28%│
│  🏠 주거비       380,000  21%│
│  🚗 차계부       245,000  13%│
│  📱 구독관리      89,000   5%│
│  🎊 경조사       100,000   5%│
├──────────────────────────────┤
│  최근 내역                    │
│  03/09 점심 외식    -12,000  │
│  03/09 주유 (차계부) -82,000 │
│  03/08 넷플릭스(구독) -17,000│
│  03/08 ○○○ 결혼식   -50,000 │
│  ...더보기                    │
└──────────────────────────────┘
```

**구성 요소:**
- 월 선택기 (좌우 스와이프 또는 드롭다운)
- 수입/지출/잔액 요약 카드
- 목표 달성률 프로그레스 바 (설정한 월 목표 저축액/저축률 기준)
- 소비성 지출 TOP 5 (차계부·구독·경조사 포함 통합)
- 최근 내역 리스트 (영역 태그 뱃지 표시)

### 1-2. 입력 화면

```
┌──────────────────────────────┐
│  지출 입력                    │
├──────────────────────────────┤
│  금액: ₩ [          12,000] │
│                              │
│  날짜: [2026-03-09]     📅  │
│  영역: [일반 ▼]              │
│        일반 / 차계부 /        │
│        구독관리 / 경조사      │
│                              │
│  결제수단: [삼성신용 ▼]       │
│  대분류:   [식비 ▼]          │
│  소분류:   [외식/배달 ▼]      │
│  내용:     [점심 김치찌개]    │
│  할인:     ₩ [         0]   │
│  비고:     [              ]  │
│                              │
│  [저장]              [취소]  │
└──────────────────────────────┘
```

**특징:**
- 영역 선택 시 해당 영역 전용 입력 필드로 전환
- 대분류 선택 시 소분류 자동 필터링 (항목 시트 기반)
- 결제수단은 설정에서 커스텀 가능
- 할인 필드로 실지출 자동 계산 (결제금액 - 할인 = 실지출)

### 2-2. 차계부 전용 입력

```
┌──────────────────────────────┐
│  차계부 지출 입력              │
├──────────────────────────────┤
│  유형: [주유 ▼]               │
│   주유 / 정비 / 보험 / 세금 / │
│   톨비 / 주차 / 세차 / 할부 / │
│   기타                        │
│                              │
│  ── 주유 상세 ──              │
│  금액:    ₩ [82,000]         │
│  주유량:  [50] L              │
│  단가:    ₩1,640 (자동계산)   │
│  주행거리: [45,230] km        │
│  주유소:  [○○주유소]          │
│  결제수단: [토스체크 ▼]       │
│  날짜:    [2026-03-09]       │
│                              │
│  [저장]              [취소]  │
└──────────────────────────────┘
```

**차계부 전용 카테고리:**

| 대분류 | 소분류 |
|--------|--------|
| 유류비/충전비 | 휘발유, 경유, LPG, 전기충전 |
| 정비/수리 | 엔진오일, 타이어, 브레이크, 일반정비, 사고수리 |
| 보험 | 자동차보험, 운전자보험 |
| 세금 | 자동차세, 환경개선부담금 |
| 할부 | 차량할부금, 리스료 |
| 톨비 | 하이패스, 톨게이트 |
| 주차 | 주차비, 월주차 |
| 세차 | 자동세차, 손세차, 실내세차 |
| 기타 | 용품구매, 기타 |

## 4. 경조사 영역

```
┌──────────────────────────────┐
│  🎊 경조사          2026년   │
├──────────────────────────────┤
│  올해 요약                    │
│  ┌───────────┬───────────┐  │
│  │  보낸 금액  │  받은 금액 │  │
│  │  850,000   │  200,000  │  │
│  └───────────┴───────────┘  │
│  순지출: -650,000            │
│                              │
│  유형별 현황                  │
│  💒 결혼  5건  -350,000     │
│  🕯 장례  2건  -200,000     │
│  🎂 돌잔치 1건  -50,000     │
│  🎁 기타  3건  -250,000     │
│  📩 받은것 2건  +200,000    │
│                              │
│  다가오는 일정                │
│  03/15 ○○○ 결혼식  예상 5만  │

│                              │
│  최근 내역                    │
│  03/08 ○○○ 결혼식  -50,000  │
│  02/22 ○○○ 장례식 -100,000  │
│  02/14 ○○○ 결혼식  -50,000  │
│  01/15 설날 세뱃돈  +200,000 │

```

### 4-2. 경조사 입력

```
┌──────────────────────────────┐
│  경조사 기록                  │
├──────────────────────────────┤
│  구분: [보낸 것 ▼]           │
│        보낸 것 / 받은 것      │
│                              │
│  유형: [결혼 ▼]              │
│   결혼 / 장례 / 돌잔치 /     │
│   생일 / 집들이 / 개업 /     │
│   명절 / 기타                 │
│                              │
│  대상: [○○○]                 │
│  관계: [대학동기 ▼]          │
│   가족 / 친척 / 직장동료 /   │
│   대학동기 / 고교동기 /       │
│   지인 / 기타                 │
│                              │
│  금액:   ₩ [50,000]         │
│  날짜:   [2026-03-08]       │
│  장소:   [○○웨딩홀]          │
│  결제수단: [현금 ▼]          │
│  메모:   [              ]    │
│                              │
│  [저장]              [취소]  │
└──────────────────────────────┘
```

**경조사 관계 태그 시스템:**
- 대상자별 누적 기록 자동 관리 (○○○에게 총 얼마 보냈고, 받았는지)
- 관계별 평균 금액 통계
- 같은 사람에게 여러 번 기록 시 자동 연결

- 카테고리별 정산도 가능 (식비는 균등, 용돈은 각자 등)
