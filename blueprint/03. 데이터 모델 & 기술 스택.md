## 1. 핵심 데이터 모델

### 1-1. 회원 (User)

| 필드 | 타입 | 설명 |
|------|------|------|
| id | UUID | PK |
| email | VARCHAR | 이메일 (로그인 ID) |
| nickname | VARCHAR | 닉네임 (가계부 내 표시명) |
| profile_image | VARCHAR | 프로필 이미지 URL (nullable) |

| role_in_group | VARCHAR | OWNER / MEMBER |
| default_payment_method_id | UUID | 기본 결제수단 FK (nullable) |
| created_at | TIMESTAMPTZ | 가입일시 |
| last_login_at | TIMESTAMPTZ | 최근 로그인 |
| status | VARCHAR | ACTIVE / DORMANT / WITHDRAWN |

### 1-2. 가족 그룹 (family_group)

| 필드 | 타입 | 설명 |
|------|------|------|
| id | UUID | PK |
| name | VARCHAR | 그룹명 |
| invite_code | VARCHAR(6) | 초대 코드 |
| invite_code_expires_at | TIMESTAMPTZ | 초대 코드 만료일시 |
| owner_id | UUID | FK → user |
| created_at | TIMESTAMPTZ | 생성일시 |

**가족 그룹 동작:**
- OWNER가 초대 코드 발급 → 구성원이 코드로 참여
- 그룹 내 모든 거래 통합 조회, 작성자 자동 기록
- OWNER: 그룹 설정, 구성원/카테고리 관리
- MEMBER: 본인 작성 CRUD + 전체 조회

```

       │              │
       │              ├── 1:1 ──→ car_expense_detail
       │              ├── 1:1 ──→ ceremony_event
       │              └── N:1 ──→ payment_method
       │
       ├── 1:N ──→ payment_method (개인 소유)
       ├── 1:N ──→ subscription
       ├── 1:N ──→ ceremony_person
       ├── 1:N ──→ budget
       ├── 1:N ──→ goal
       ├── N:1 ──→ family_group
       └── 1:N ──→ category_config (owner_type=USER)

family_group ──→ 1:N ──→ user (구성원)
               └─→ 1:N ──→ category_config (owner_type=FAMILY_GROUP)
```

### 3-3. 정산 SQL 쿼리 (참고)

개인별 카드 사용 집계는 PostgreSQL에서 간단하게 처리:

```sql
-- 가족 그룹 내 구성원별 + 결제수단별 월간 집계
SELECT
  u.nickname,
  pm.name AS card_name,
  pm.type AS card_type,
  SUM(t.actual_amount) AS total_amount,
  COUNT(*) AS tx_count
FROM transaction t
JOIN "user" u ON t.user_id = u.id
JOIN payment_method pm ON t.payment_method_id = pm.id
WHERE t.family_group_id = :group_id
  AND t.date BETWEEN :start_date AND :end_date
  AND t.type = 'EXPENSE'
GROUP BY u.id, u.nickname, pm.id, pm.name, pm.type
ORDER BY u.nickname, total_amount DESC;
```

### 3-4. 정산 기능 동작 방식

- 결제수단(payment_method)은 각 유저 개인 소유
- 거래 기록 시 본인의 결제수단만 선택 가능
- 가족 모드에서 "카드 사용 현황" 탭 → 구성원별 카드별 사용 금액 자동 집계
- "월간 정산" → 균등 분할 기준 차액 자동 계산
- 정산 비율 커스텀 가능 (예: 6:4, 7:3 등)
- 월별 추이 비교 가능

